# Element-FX

A `JavaFX` version of `element-plus`.

`JavaFX` 版本的 `element-plus`。

## 目标 Target

移植 [Element-Plus](https://element-plus.org) UI库到 `JavaFX` 平台，并提供桌面端优化。

使用**Java 11**开发，兼容**Java 8**及以上版本。

## 模块 Module

### Lite

提供样式资源，无交互代码

### Demo

提供样例
